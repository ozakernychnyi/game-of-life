#### **Game of life**

This game variant was checked to run under Mac. For checking program execution use please dedicated terminal window, don't use IDE built-in terminal please.
To execute game navigate via terminal to game source directory and run `go run main.go`. Execution was verified under `bash` and `zsh` terminals.
Also in zip archive you can find `game_of_life` binary, which should be able to run on `Mac (Darwin)` with arch `X86_64`.
